# # Contributing Guidelines

Thanks for helping improve the analysis of iCUE 5.x issues!

## How to Contribute
- Submit logs that demonstrate the write loop  
- Provide Process Monitor traces  
- Confirm CPUID service startup failures  
- Submit patches or documentation improvements  

## Submitting Issues
When filing an issue, include:

- Windows version  
- iCUE version  
- Hardware (coolers, controllers, fans)  
- Steps to reproduce  
- Logs (Process Monitor, Event Viewer, iCUE logs)  

## Pull Requests
Ensure:

- Markdown is clean  
- Technical accuracy  
- No personal data in logs
